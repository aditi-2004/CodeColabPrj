Frontend (React.js)
Authentication: Login, Signup, Profile
Dashboard for managing projects
Project page with details & members
Collaborative real-time code editor
Live chat for team communication
Routing via React Router
API integration using Axios/Fetch

Backend (Node.js + Express.js)
JWT authentication + bcrypt password hashing
REST APIs: Users, Projects, Chat, Code Collaboration
WebSockets (Socket.io) for real-time code & chat sync
MongoDB (Mongoose) for Users, Projects, Chat messages
Middleware: express.json(), Auth checks, Error handling
